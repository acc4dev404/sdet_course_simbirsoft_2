# Тест-кейсы для банковского приложения 

## Объект тестирования
https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager

## Тест-кейс 1: Создание клиента (Add Customer)

**ID:** TC001  
**Приоритет:** High  
**Предусловия:**
- Открыта страница управления клиентами

**Шаги:**
1. Нажать кнопку "Add Customer"
2. В поле "First Name" ввести сгенерированное имя
3. В поле "Last Name" ввести "TestUser"
4. В поле "Post Code" ввести 10-значный номер
5. Нажать кнопку "Add Customer"

**Ожидаемый результат:**
- Появляется alert с сообщением "Customer added successfully"
- После принятия alert клиент добавляется в таблицу Customers

## Тест-кейс 2: Сортировка клиентов по имени (First Name)

**ID:** TC002  
**Приоритет:** Medium  
**Предусловия:**
- Открыта страница управления клиентами
- В таблице присутствует несколько клиентов

**Шаги:**
1. Открыть вкладку "Customers"
2. Нажать на заголовок столбца "First Name"
3. Запомнить порядок клиентов

**Ожидаемый результат:**
- Клиенты отсортированы по имени в алфавитном порядке (A-Z)

## Тест-кейс 3: Удаление клиента

**ID:** TC003  
**Приоритет:** High  
**Предусловия:**
- Открыта страница управления клиентами
- В таблице присутствует несколько клиентов

**Шаги:**
1. Открыть вкладку "Customers"
2. Получить список всех имен клиентов из таблицы
3. Вычислить среднюю длину имен
4. Найти клиента с длиной имени, наиболее близкой к средней
5. Нажать кнопку "Delete" для найденного клиента

**Ожидаемый результат:**
- Выбранный клиент удаляется из таблицы
- Количество клиентов уменьшается на 1
